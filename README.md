The largest digital memorial directory in Southern Africa. Discover, locate, and honor loved ones through an interactive map and comprehensive memorial listings.

## Features
- **Interactive Map**: Browse memorials with GPS coordinates via Leaflet
- **Photo-Rich Listings**: Each memorial includes photos and scenic views
- **Life Stories**: Read detailed biographies and personal tributes
- **Visit Planning**: Get directions and plan visit routes
- **Pricing Tiers**: Free, Premium (R99/yr), and Family (R249/yr) plans
- **Testimonials**: Real stories from families who found connection
- **Responsive Design**: Works seamlessly on all devices

## How to Run
Open `index.html` in any modern web browser. No server required.

```bash
# Clone the repo
git clone https://github.com/CodeBreweryZA/Umzukulu-finder.git

# Open directly in browser
cd Umzukulu-finder
start index.html
```

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla ES6+)
- Leaflet.js for mapping
- Font Awesome 6
- Google Fonts (Inter, Playfair Display)
- LocalStorage for data persistence
